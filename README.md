### 📁 Мои проекты:
1. Финансовые показатели компании в сфере красоты (https://github.com/JaBaJoker/Financial_Indicators_Beauty_Company)
- Задачи:
1) Разработка единого дашборда для мониторинга ключевых финансовых метрик (выручка, продажи, средняя цена, % выкупа) в разрезе брендов, категорий и стран.
2) Внедрение автоматизированного отслеживания динамики показателей месяц к месяцу (MoM) с сигнальной системой отклонений.
3) Анализ зависимости процента выкупа от цены товара для выявления позиций, требующих ценовой коррекции (скидки).
4) Сегментация ассортимента для выявления неэффективных продуктов и обоснования решений по их исключению или продвижению.
- Стек технологий:
DataLens (дашборд)
- Результаты:
1) Создал интерактивный дашборд с фильтрами по брендам/категориям, позволяющий отслеживать выручку, продажи и % выкупа в динамике.
2) Настроил автоматический расчет план/факт по выручке (цель 5% MoM) и % выкупа (цель 95%) с подсветкой критических отклонений.
3) Построил матрицу "Цена — % выкупа", выявил дорогие товары с низкой конверсией и предложил рекомендации по скидкам.
4) Обеспечил актуализацию данных без ручного сбора (консолидация из двух источников: 2 месяца + полугодовая выгрузка).
